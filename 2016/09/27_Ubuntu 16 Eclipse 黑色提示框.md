用 sudo 打开:  
> /usr/share/themes/<主题名, 如 Ambiance>/gtk-3.0/gtk-main.css  

把
``` css
@define-color tooltip_bg_color #000000;
@define-color tooltip_fg_color #ffffff;
```

改为
``` css
@define-color tooltip_bg_color #f2edbc;
@define-color tooltip_fg_color #000000;
```

保存 并 重启电脑 即可.
