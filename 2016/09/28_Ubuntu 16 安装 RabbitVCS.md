官网安装教程  
> http://wiki.rabbitvcs.org/wiki/install/ubuntu  

但是 我的 `Ubuntu 16` 安装 `rabbitvcs-nautilus3` 会失败.  
所以用了下面的语句 作替换:  
``` bash
sudo apt update
sudo apt install rabbitvcs-nautilus rabbitvcs-cli rabbitvcs-gedit rabbitvcs-core
```
``` bash
nautilus -q
nautilus
```
