# SVN
**svn 核心 Ubuntu 自带, 所以不用刻意下载**  

好用的 GUI 工具 RapidSVN:  
http://www.rapidsvn.org/index.php/Main_Page  
``` bash
sudo apt install rapidsvn
``` 

# Git
先要安装:
``` bash
sudo apt install git
```

GUI 工具 Git 官网推荐了几个:  
> https://git-scm.com/download/gui/linux

我比较喜欢 `GitKraken` 和 `git-cola` 和 `GitEye`.
