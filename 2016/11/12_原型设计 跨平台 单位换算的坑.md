# DPI
```
- MacOS/iOS 默认 1:1 DPI = 72
- Windows 默认 1:1 DPI = 96
- Ubuntu 默认 1:1 DPI = 90~98(待核实)
- Android 默认 1:1 DPI = 由设备/ROM决定
```

# DPI 常量
```
- ldpi = 90 ~ 120
- mdpi = 160
- hdpi = 240
- xhdpi = 320
- xxhdpi = 480
- xxxhdpi = 640
```

# 术语
```
- pt(point): 磅
- px(pixel): 像素
- inch(in): 英寸
- dip/dp(device-independent pixel): 设备独立像素
- sp(scale-independent pixel): 比例独立像素
- density: 密度
- resolution: 分辨率
- physical: 物理的
- softwarcal: 软件层的
======
- PPI(pixels per inch): 每英寸像素数
- DPI(dots per inch): 每英寸点数
- BIR(base inch ratio): 基本英寸比例
- DPS(device physical size (unit in inch)): 设备物理尺寸
- TSS(text size scale): 文本字号比例
- 
```

# 换算
```
- dpi ≈ ppi
- density = DPI / mdpi
- BIR = 1/72
```

```
- 1 pt = BIR in
- 1 px = 1 pt * DPI * BIR
- 1 dp = 1 px * density
- 1 sp = 1 dp * TSS
```
