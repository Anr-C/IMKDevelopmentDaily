# DPI
```
- MacOS/iOS 默认 1:1 DPI = 72
- Windows 默认 1:1 DPI = 96
- Ubuntu 默认 1:1 DPI = 90~98(待核实)
- Android 默认 1:1 DPI = 由设备/ROM决定
```

# DPI 常量
```
- ldpi = 90 ~ 120
- mdpi = 160
- hdpi = 240
- xhdpi = 320
- xxhdpi = 480
- xxxhdpi = 640
```

# 术语
```
- point(pt): 磅
- pixel(px): 像素
- inch(in): 英寸
- device-independent pixel(dip/dp): 设备独立像素
- scale-independent pixel(sp): 比例独立像素
- density: 密度
- resolution: 分辨率
- physical: 物理的
- softwarcal: 软件层的
```

# 术语 (缩写)
```
- PPI: pixels per inch
- DPI: dots per inch
- BIR: base inch ratio
- DPS: device physical size (unit in inch)
- TSSR: text size scale ratio
```

# 换算
```
- dpi ≈ ppi
- density = DPI / mdpi
- BIR = 1/72
```

```
- 1 pt = BIR in
- 1 px = 1 pt * DPI * BIR
- 1 dp = 1 px * density
- 1 sp = 1 dp * TSSR
```
