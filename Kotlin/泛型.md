PECS

|    方向   |     Java    | Kotlin |      |       |                      | 备注                       |
| :------: | :---------: | :----: | ---- | ----- | -------------------- | ------------------------- |
| 上界通配符 | ? extends T |  out   | 协变 | 生产者 | 只能读取, 不能修改(放置) | 不含 clear() 之类          |
| 下界通配符 | ? super T   |   in   | 逆变 | 消费者 | 只能修改(放置), 不能读取 | 不含 取出 Object 再强转 之类 |
